### Real-time вариант Б1

Если говорить не про асинхронные ролики, а про "Characters API вариант Б1" как про real-time видео-чат с игроком, то здесь ограничения становятся еще неприятнее. Для обычной разговорной коммуникации люди начинают замечать лаг уже в районе 100-120 мс, а нормальный разговор начинает ломаться примерно на 250-300 мс. При этом Twilio для каскадного voice-agent pipeline в конце 2025 года дает стартовый benchmark mouth-to-ear turn gap примерно 1115 мс, с upper limit около 1400 мс; отдельно STT target около 350 мс, LLM time-to-first-token около 375 мс и TTS time-to-first-byte около 100 мс. Это уже терпимо для call-center bot, но слабо похоже на "живой relationship manager face-to-face". Для видео-аватара сверху еще добавляется lip-sync/render overhead. citeturn31search0turn32view0

Инфраструктурно real-time опирается на WebRTC/SFU.
