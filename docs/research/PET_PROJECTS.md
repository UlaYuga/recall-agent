# Pet-проекты Александра Уланова

GitHub: https://github.com/UlaYuga

## 1. FKNG MARK - автономный маркетинг-движок (флагман)

**Стэк:** Next.js, Postgres, Claude Agent SDK
**Демо:** https://ulayuga.github.io/launch-engine-demo/

AI-агент автономно проходит цикл: ресерчит тему → пишет контент → публикует в 4 канала → собирает метрики → через 48 часов сам решает продолжать кампанию или резать.

**Что показывает:** способность собрать end-to-end agentic-систему с автономным принятием решений по метрикам. Уровень AI Implementation Manager не на словах, а в проде.

## 2. Гадалка - mini-app для MAX

**Стэк:** MAX mini-app платформа, DeepSeek API, ЮKassa
**Демо:** https://ulayuga.github.io/Esoteric_MAX_Public/

4 эзотерических AI-инструмента с живыми ответами через DeepSeek (не шаблоны). Монетизация через ЮKassa.

**Что показывает:** работа с консьюмерским продуктом, интеграция LLM в прод, понимание UX, монетизация.

## 3. FarmFun - бот-кликер для MAX

**Стэк:** Node.js, SQLite, Yandex Cloud
**Демо:** https://ulayuga.github.io/farmfun-chat-demo/

Кликер с виральными механиками: реферальная система, PvP-кражи урожая, помощь друзьям.

**Что показывает:** разработка социальных и retention-механик, инфра на облаке, понимание engagement-петель (релевантно для iGaming CRM/Retention ролей).

## 4. Незнакомец - визуальная новелла для VK

**Стэк:** VK mini-app, Midjourney (иллюстрации), ElevenLabs (озвучка)
**Демо:** https://ulayuga.github.io/Neznakomets-web/

Интерактивная новелла, 200 сцен с ветвлением, полный AI-pipeline производства.

**Что показывает:** AI-производство контента в масштабе, продакшн-процессы, работа с креативными ассетами.

## 5. ULA Lab - сервис для кофе-экспериментов

**Стэк:** веб-сервис, кастомная логика моделирования инфузий
**Демо:** https://ulalab.online/

Помогает создавать infused-кофе - расчет пропорций, длительности, конфигурации экспериментов.

**Что показывает:** строю инструменты под собственные потребности и масштабирую под других пользователей.

## 6. HH Vacancy Parser (рабочий инструмент)

**Стэк:** Python, hh.ru API + browser automation fallback, pandas, xlsx
**Что делает:** парсит hh.ru по 130+ запросам в 9 кластерах с round-robin интерливингом, классифицирует вакансии, скорит, парсит зарплаты с конвертацией gross/net, выдает ranked xlsx + role_clusters.md + raw json.

**Что показывает:** автоматизация рабочих процессов через скрипты, понимание ETL-логики, работа с API и парсингом данных.
