# Vendor Integration Plan

See [PM Delivery Artifacts](PM_DELIVERY_ARTIFACTS.md) for the full source pack.

## MVP Vendors

- Runway API: video generation and TTS.
- Anthropic Claude: script, CTA, and visual prompt generation.
- Telegram Bot API: PoC delivery.
- Vercel: landing and dashboard hosting.
- Railway or Render: FastAPI backend hosting.

## Roadmap

- PostgreSQL instead of SQLite.
- S3 or GCS for media storage.
- Redis queue for async jobs.
- SendGrid email adapter.
- WhatsApp Business and SMS adapters where consent and templates are available.
