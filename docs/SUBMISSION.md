# Runway Submission

## Project

Recall - AI Retention Agent for Consumer Subscription Products.

## Short Description

Recall is an event-driven retention agent that monitors user activity, identifies churn risk, generates personalized motion-graphics video messages with Runway API, routes every message through human approval, delivers through a Telegram PoC adapter, and tracks conversion through a campaign landing page.

## Built With

- Runway Gen-4.5
- Runway text-to-speech
- FastAPI
- Next.js
- Telegram Bot API
- SQLite mock CRM/event bus

## Demo Proof

- Working approval dashboard.
- End-to-end campaign path.
- Generated personalized video postcard.
- Tracking and ROI dashboard.
