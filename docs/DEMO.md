# Demo

## Three-Minute Walkthrough

1. Start local services with `make dev`.
2. Seed mock users with `make seed`.
3. Trigger `/agent/scan`.
4. Open dashboard approval queue.
5. Approve one campaign draft.
6. Generate or use fallback video asset.
7. Send via Telegram adapter.
8. Open `/r/[campaign_id]` landing page.
9. Trigger play, click, and mock deposit tracking.
10. Show metrics dashboard and ROI simulation.
