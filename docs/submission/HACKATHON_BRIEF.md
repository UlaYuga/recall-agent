## Что строить

> "Build the best [agent / app / tool / creative workflow] with Runway's API"

Хакатон про **agents и applications, которые create, manipulate or orchestrate media using AI**. Думать **за пределами single API calls** - системы, которые chain together creative decisions, respond to inputs, produce something people actually want to use.

Гайдлайны от Runway (suggestions, not constraints):
- Autonomous agent that generates and iterates on video content based on a brief
- Tool that lets non-technical users direct AI video production через natural language
- Workflow that uses Runway API to automate real media production pipeline
- Interactive app built on Runway Characters API

**Единственное правило:** проект должен использовать Runway API.

## Критерии оценки

| Критерий | Что смотрят |
|---|---|
| **Creativity** | Решает ли проект реальную проблему или explores интересный use case |
| **Technical depth** | Насколько хорошо проект leverages Runway API. Идет ли beyond basic usage |
| **Impact** | Может ли это быть использовано real users/customers. Может ли стать real product |
| **Polish** | Working demo, не concept. Feels end-to-end |
